gprof_nn
========

This package implements the GPROF-NN 1D and GPROF-NN 3D retrievals for the passive microwave
observations from the Global Precipitation Measurement (GPM) mission. It consists of two
principal parts:

1. The ``gprof_nn`` command line application, which provides data processing, training and
   retrieval functionality.
2. The ``gprof_nn`` Python package, which provides functions for the processing GPM-related
   data.
